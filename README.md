## 🚀 Featured Project

### 🖼️ MNIST Image Classifier (Softmax Neural Network)

Located in:
```
02_advanced_learning_algorithms/image_classifier_mnist/
```

### 🔍 What it does:
- Classifies handwritten digits (0–9)
- Uses a neural network with softmax output
- Outputs probability distribution per class

### 🧠 Concepts used:
- Softmax function
- Cross-entropy loss
- Neural networks
- Argmax classification

### 📊 Example Output:
- Test accuracy: ~97–98%
- Probabilistic predictions per image
